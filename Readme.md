# SubSentry - Subscription Tracker

SubSentry is a web application designed to help users efficiently track their subscriptions and avoid missing renewal dates. It allows managing multiple subscriptions with details like cost, billing cycle, and renewal dates.

## Features

- Add new subscriptions with service name, cost, billing cycle, next renewal date, and category
- View all active subscriptions in a clean dashboard layout
- See total subscription count, monthly cost, and yearly cost at a glance
- Highlight subscriptions renewing within 7 days for timely reminders
- Edit or delete existing subscriptions easily
- Responsive design with light and dark mode support

## Technologies Used

- HTML5
- CSS3 (including CSS variables for theming)
- JavaScript for subscription management and UI functionality

## Usage

1. Clone the repository:
git clone https://github.com/your-username/SubSentry.git

2. Open `index.html` in your favorite browser.
3. Add, edit, or delete subscriptions as needed to track your recurring expenses.

## Future Enhancements

- Add user authentication to save subscriptions in the cloud
- Notifications for upcoming renewals
- Export subscription data to CSV or PDF
- Mobile app version for on-the-go management
